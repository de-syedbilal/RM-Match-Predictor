How to run this locally
If you’re curious and want to run the code on your own machine, here is how I set it up:

1) Clone this repo to your computer.
2) Open your terminal in the project folder.
3) Run pip install -r requirements.txt to get all the libraries.
4) Once that’s done, type streamlit run MatchCalc.py and it should open right up in your browser.
